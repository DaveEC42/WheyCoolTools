# WheyCoolTools
Various C# classes to assist developers.

Currently included:
- ConvertMeasurement v0.3
  - Provides a simple way to convert between common measurements (e.g.: Kilometers to Miles, Fahrenheit to Celsuis)
- OperationResult v1.0
  - Message wrapper class that has a boolean property and a List<string> property. Example of use: Validating an operation and providing messages based on success or failure.
